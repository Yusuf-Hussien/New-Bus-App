## Overview

**New Bus App** is a university team project designed to solve a
real-world transportation challenge by providing a smart bus management
and booking system.\
The system aims to digitalize bus operations, improve user experience
for passengers, and provide administrators with clear control over
buses, routes, and finances.

This project is developed collaboratively by a multidisciplinary team,
combining backend, frontend, DevOps, UI/UX, and system analysis skills.

------------------------------------------------------------------------

## Problem Statement

Traditional bus systems often suffer from: - Manual processes - Poor
user experience - Lack of transparency in payments and subscriptions -
Limited financial reporting for administrators

**New Bus App** addresses these challenges by offering a modern,
scalable, and extensible solution.
